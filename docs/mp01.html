#' Acquire IPEDS Data for MP#01
#' 
#' This function will acquire and standardize all data for MP#01
#' from IPEDS (https://nces.ed.gov/ipeds/datacenter/DataFiles.aspx)
#' 
#' We're starting in 2010 as the data seems to be reasonably complete 
#' after that point. 
acquire_ipeds_data <- function(start_year=2010, end_year=2024){
    data_dir <- file.path("data", "mp01")
    
    if(!dir.exists(data_dir)){
        dir.create(data_dir, showWarnings = FALSE, recursive = TRUE)
    }
    
    YEARS <- seq(start_year, end_year)
    
    EFA_ALL <- map(YEARS, function(yy){
        if(yy <= 2022){
            ef_url <- glue("https://nces.ed.gov/ipeds/datacenter/data/EF{yy}A.zip")
        } else {
            ef_url <- glue("https://nces.ed.gov/ipeds/data-generator?year={yy}&tableName=EF{yy}A&HasRV=0&type=csv")
        }
        
        ef_file <- file.path(data_dir, glue("ef{yy}a.csv.zip"))
        
        if(!file.exists(ef_file)){
            message(glue("Downloading Enrollment Data for {yy}"))
            download.file(ef_url, destfile = ef_file, quiet = TRUE, mode = "wb")
        }
        
        read_csv(ef_file, show_col_types = FALSE) |>
            mutate(
                year = yy,
                enrollment_m_aian = EFAIANM,
                enrollment_f_aian = EFAIANW,
                enrollment_m_asia = EFASIAM,
                enrollment_f_asia = EFASIAW,
                enrollment_m_bkaa = EFBKAAM,
                enrollment_f_bkaa = EFBKAAW,
                enrollment_m_hisp = EFHISPM,
                enrollment_f_hisp = EFHISPW,
                enrollment_m_nhpi = EFNHPIM,
                enrollment_f_nhpi = EFNHPIW,
                enrollment_m_whit = EFWHITM,
                enrollment_f_whit = EFWHITW,
                enrollment_m_2mor = EF2MORM,
                enrollment_f_2mor = EF2MORW,
                enrollment_m_unkn = EFUNKNM,
                enrollment_f_unkn = EFUNKNW,
                enrollment_m_nral = EFNRALM,
                enrollment_f_nral = EFNRALW
            ) |>
            filter((EFALEVEL %in% c(2, 12)) | (LINE %in% c(1, 15))) |>
            mutate(
                level = case_when(
                    EFALEVEL == 2 ~ "all undergrad",
                    EFALEVEL == 12 ~ "all graduate",
                    LINE %in% c(1, 15) ~ "first year undergrad"
                )
            ) |>
            select(
                institution_id = UNITID,
                year,
                level,
                starts_with("enrollment_")
            ) |>
            group_by(institution_id, year, level) |>
            summarize(across(starts_with("enrollment_"), sum), .groups = "drop")
    }) |> bind_rows()
    
    DESC_ALL <- map(YEARS, function(yy){
        if(yy <= 2022){
            hd_url <- glue("https://nces.ed.gov/ipeds/datacenter/data/HD{yy}.zip")
        } else {
            hd_url <- glue("https://nces.ed.gov/ipeds/data-generator?year={yy}&tableName=HD{yy}&HasRV=0&type=csv")
        }
        
        hd_file <- file.path(data_dir, glue("hd{yy}.csv.zip"))
        
        if(!file.exists(hd_file)){
            message(glue("Downloading Institutional Descriptions for {yy}"))
            download.file(hd_url, destfile = hd_file, quiet = TRUE, mode = "wb")
        }
        
        suppressWarnings(
            read_csv(
                hd_file,
                show_col_types = FALSE,
                locale = locale(encoding = if_else(yy == 2024, "utf-8", "windows-1252"))
            ) |>
            mutate(year = yy) |>
            select(
                institution_id = UNITID,
                institution_name = INSTNM,
                state = STABBR,
                year
            )
        )
    }) |> bind_rows()
    
    inner_join(
        EFA_ALL,
        DESC_ALL,
        join_by(institution_id == institution_id, year == year)
    )
}

Task 7: College Newspaper Op-Ed

Diversity at Baruch College: A Reflection of Our City

As president of Baruch College, I believe that diversity is one of the greatest strengths of our institution. Baruch has long served students from many different backgrounds, reflecting the multicultural nature of New York City itself. A diverse campus allows students to learn from peers with different perspectives and experiences, which strengthens the overall educational environment.

For this analysis, diversity is defined as the fraction of undergraduate students who are not White and not Asian American. This measure helps highlight the representation of historically underrepresented groups within the student body.

Baruch College has a large and diverse student population. In 2024, the college enrolled approximately 20,081 total students. The undergraduate population makes up the majority of this enrollment and represents a wide variety of racial and ethnic backgrounds.

Looking specifically at the incoming class, Baruch enrolled about 2,532 first-year undergraduate students in 2024. The size of the entering class has remained relatively stable over recent years, indicating consistent demand for a Baruch education.

Over time, the demographic composition of the student body has continued to evolve. As New York City becomes increasingly diverse, the student population at Baruch reflects those broader demographic changes. This trend highlights the important role that public universities play in providing access to higher education for students from many different communities.

Comparing Baruch with other public institutions can also provide useful context. For example, large public universities in California have experienced similar demographic changes in recent years. Because California universities operate under different policy environments, they serve as a useful benchmark when examining whether demographic changes are driven by institutional policy or by broader social and demographic trends.

Overall, diversity strengthens the learning environment at Baruch College. Students benefit from engaging with classmates who bring different cultural backgrounds and life experiences to the classroom. By continuing to support access to education and maintaining an inclusive campus environment, Baruch can remain a place where students from many communities can learn, grow, and succeed.

Entropy Analysis

Entropy is a statistical measure used to evaluate diversity in a population. In this context, entropy measures how evenly different racial and ethnic groups are represented among students. A higher entropy value indicates a more diverse student population because enrollment is distributed more evenly across groups.

To measure diversity using entropy, we first calculate the proportion of students in each racial and ethnic group and then apply the entropy formula.

IPEDS_entropy <- IPEDS |>
  filter(level == "all undergrad") |>
  mutate(
    aian = enrollment_m_aian + enrollment_f_aian,
    asia = enrollment_m_asia + enrollment_f_asia,
    bkaa = enrollment_m_bkaa + enrollment_f_bkaa,
    hisp = enrollment_m_hisp + enrollment_f_hisp,
    nhpi = enrollment_m_nhpi + enrollment_f_nhpi,
    whit = enrollment_m_whit + enrollment_f_whit,
    other = enrollment_m_2mor + enrollment_f_2mor +
            enrollment_m_unkn + enrollment_f_unkn +
            enrollment_m_nral + enrollment_f_nral
  ) |>
  rowwise() |>
