Introduction

The Olympic Games remain one of the world’s most significant international sporting events, attracting global audiences, corporate sponsorships, and national attention. With Los Angeles set to host the 2028 Summer Olympics, expectations surrounding Team USA’s performance are especially high. Historically, host nations often experience improved medal performance due to factors such as home-field advantage, increased investment in athlete development, and automatic qualification opportunities in selected events.

In this mini-project, I analyze historical Olympic medal results using data scraped directly from Olympedia. The analysis focuses on three major factors that may influence Team USA’s 2028 Olympic performance:

  1. Team USA’s historical baseline medal performance
  2. The host-country advantage effect
  3. The impact of newly introduced Olympic sports

To complete this analysis, I use modern data analytics techniques including web scraping, data cleaning, exploratory data analysis, statistical inference, and Monte Carlo simulation modeling. The final goal is to forecast Team USA’s expected medal performance in the 2028 Los Angeles Olympics and summarize the findings in a concise fundraising-style report suitable for stakeholders and sponsors.

Data Acquisition and Preparation

This section builds the core Olympic Games data set directly from Olympedia. I first scrape the main Olympic editions page, then extract each Games’ season, year, Olympiad number, Olympedia edition ID, host city, and host country code. The host country code is extracted from the flag image URL, which is necessary because the assignment requires information beyond the plain text returned by html_table().

Cached Olympedia Reader

To avoid repeatedly downloading the same pages from Olympedia, I use the cached reader recommended in the assignment instructions. The first time a page is requested, it is saved locally; later requests read the saved copy.

Code 
slow_download <- purrr::slowly(
  download.file,
  purrr::rate_delay(pause = 5)
)

read_olympedia <- function(url, ...) {
  
  if (stringr::str_starts(url, "/")) {
    url <- stringr::str_remove(url, "/")
  }
  
  cache_dir <- fs::path("data", "mp04", "olympedia_cache")
  
  if (!fs::dir_exists(cache_dir)) {
    fs::dir_create(cache_dir, recurse = TRUE)
  }
  
  cache_path <- fs::path(
    cache_dir,
    url |>
      as.character() |>
      stringr::str_replace_all("/", "_")
  )
  
  if (!fs::file_exists(cache_path)) {
    src_url <- paste0("https://www.olympedia.org/", url)
    
    slow_download(
      src_url,
      cache_path,
      mode = "wb"
    )
  }
  
  rvest::read_html(cache_path)
}

parse_game_row <- function(this_row) {
  
  cells <- this_row |> html_elements("td")
  
  edition_url <- this_row |>
    html_element("a[href^='/editions/']") |>
    html_attr("href")
  
  host_code <- this_row |>
    html_element("img") |>
    html_attr("src") |>
    str_extract("[A-Z]{3}(?=\\.png)")
  
  tibble(
    olympiad_roman = cells |> html_element(xpath = ".") |> html_text2() |> pluck(1, .default = NA_character_),
    year = cells |> html_text2() |> pluck(2, .default = NA_character_) |> parse_number() |> as.integer(),
    city = cells |> html_text2() |> pluck(3, .default = NA_character_),
    edition_url = edition_url,
    host_code = host_code
  )
}

extract_sports <- function(edition_url) {
  
  edition_page <- read_olympedia(edition_url)
  
  links <- edition_page |>
    html_elements("a")
  
  map_dfr(links, function(this_link) {
    tibble(
      discipline = this_link |> html_text2(),
      sport_url = this_link |> html_attr("href")
    )
  }) |>
    filter(
      !is.na(sport_url),
      str_detect(sport_url, "/sports/")
    ) |>
    mutate(
      sport_url = str_remove(sport_url, "^/")
    ) |>
    distinct(sport_url, .keep_all = TRUE) |>
    select(discipline, sport_url)
}

read_medal_table <- function(sport_url) {
  
  sport_page <- read_olympedia(sport_url)
  
  medal_tables <- sport_page |>
    html_elements("table")
  
  possible_tables <- medal_tables |>
    map(html_table)
  
  medal_table <- keep(
    possible_tables,
    ~ all(c("Gold", "Silver", "Bronze") %in% names(.x))
  )
  
  if(length(medal_table) == 0) {
    return(NULL)
  }
  
  medal_table[[1]] |>
    clean_names() |>
    mutate(
      sport_url = sport_url
    )
}

clean_event_medal_table_v2 <- function(medal_item) {
  
  result_table <- medal_item$result
  
  if (is.null(result_table)) {
    return(tibble())
  }
  
  needed_cols <- c("sport_url", "gold_2", "silver_2", "bronze_2")
  
  if (!all(needed_cols %in% names(result_table))) {
    return(tibble())
  }
  
  cleaned_table <- result_table |>
    select(
      sport_url,
      gold_country = gold_2,
      silver_country = silver_2,
      bronze_country = bronze_2
    ) |>
    pivot_longer(
      cols = c(gold_country, silver_country, bronze_country),
      names_to = "medal_type",
      values_to = "competitor_code"
    ) |>
    filter(
      !is.na(competitor_code),
      str_detect(competitor_code, "^[A-Z]{3}$")
    ) |>
    mutate(
      medal_type = str_remove(medal_type, "_country")
    ) |>
    count(
      sport_url,
      competitor_code,
      medal_type,
      name = "medal_count"
    ) |>
    pivot_wider(
      names_from = medal_type,
      values_from = medal_count,
      values_fill = list(medal_count = 0)
    )
  
  if (!"gold" %in% names(cleaned_table)) cleaned_table[["gold"]] <- 0L
  if (!"silver" %in% names(cleaned_table)) cleaned_table[["silver"]] <- 0L
  if (!"bronze" %in% names(cleaned_table)) cleaned_table[["bronze"]] <- 0L
  
  cleaned_table |>
    mutate(
      gold = as.integer(gold),
      silver = as.integer(silver),
      bronze = as.integer(bronze),
      total = gold + silver + bronze
    ) |>
    select(
      sport_url,
      competitor_code,
      gold,
      silver,
      bronze,
      total
    )
}

Monte Carlo Forecasting for LA 2028

To combine the uncertainty from multiple Olympic performance effects, I use a Monte Carlo simulation approach. This method repeatedly samples from the estimated confidence intervals developed earlier in the analysis to generate a range of possible Team USA medal outcomes for the 2028 Los Angeles Olympics.
