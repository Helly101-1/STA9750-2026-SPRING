Introduction

This report examines patterns of internal migration across the United States using data from the American Community Survey (ACS). By analyzing state-to-state and metro-to-metro migration flows, we explore which states are gaining and losing residents, where New York’s migrants come from and go to, and what these trends suggest about the future distribution of political power in Congress.

Using population projections built from ACS migration data, we forecast each state’s population in 2030 and estimate how congressional seats may be reapportioned following the next decennial census. The analysis concludes with a policy proposal for how New York State could strategically attract more residents ahead of the 2032 redistricting cycle.

Data sources include ACS-1 state-level migration flows (2023–2024), ACS-5 metro-level migration flows, and state population estimates accessed via the tidycensus package.

get_state_pop <- function(year) {
  get_acs(
    variables = c(total_population = "B01003_001"),
    geography = "state",
    year = year,
    survey = "acs1",
    geometry = TRUE,
    cache_table = TRUE
  ) |>
    mutate(year = year) |>
    select(
      state_name = NAME,
      population = estimate,
      year,
      geometry
    )
}

  data_file <- file.path(data_dir, "migration_2023.xlsx")

      destfile = data_file,
      mode = "wb"
    )
  }

  states_with_dc <- c(state.name[1:8], 
                      "District of Columbia", 
                      state.name[9:50])

  # Part 1: stationary population (first 6 columns)
  stationary <- read_excel(data_file, range = "A7:F57") |>
    select(state_current = 1, pop1 = 5, pop2 = 6) |>
    filter(!is.na(state_current), !is.na(pop1)) |>
    filter(state_current %in% states_with_dc) |>
    mutate(
      population = as.integer(pop1) + as.integer(pop2),
      state_1y = state_current,
      year_current = 2023L,
      year_1y = 2022L
    ) |>
    select(state_current, state_1y, population, year_current, year_1y)

  # Part 2: flows (columns J to DG)
 flows <- read_excel(data_file, 
                    range = "J7:DG57",
                      col_names = vctrs::vec_interleave(states_with_dc, NA)) |>
    select(any_of(states_with_dc)) |>
    mutate(state_current = states_with_dc) |>
    pivot_longer(
      cols = -state_current,
      names_to = "state_1y",
      values_to = "population"
    ) |>
    filter(!is.na(population)) |>
    filter(state_current != state_1y) |>
    filter(state_current != "District of Columbia",
           state_1y != "District of Columbia") |>
    mutate(
      population = as.integer(population),
      year_current = 2023L,
      year_1y = 2022L
    ) |>
    select(state_current, state_1y, population, year_current, year_1y)

  bind_rows(stationary, flows)

metro_to_metro_migration <- function() {
  data_dir <- file.path("data", "mp03")
  data_file <- file.path(data_dir, "metro_migration.json")
  
  if(!file.exists(data_file)){
    download.file(
      "https://api.census.gov/data/2020/acs/flows?get=MOVEDIN,MOVEDOUT,FULL1_NAME,FULL2_NAME&for=metropolitan%20statistical%20area/micropolitan%20statistical%20area:*",
      destfile = data_file,
      mode = "wb"
    )
  }
  
  pull_state_from_metro <- function(metro_name){
    str_extract(metro_name, ".*, (\\S{2})[-[:alpha:]]* Metro Area", group=1)
  }
  
  raw <- read_json(data_file)
  col_names <- unlist(raw[[1]])
  rows <- raw[-1]
  
  df <- do.call(rbind, lapply(rows, function(x) {
    x[sapply(x, is.null)] <- NA
    unlist(x)
  })) |>
    as.data.frame() |>
    setNames(col_names) |>
    as_tibble() |>
    readr::type_convert()
  
  df |>
    select(metro1 = FULL1_NAME, metro2 = FULL2_NAME,
           moved_in = MOVEDIN, moved_out = MOVEDOUT) |>
    mutate(
      metro1_state = pull_state_from_metro(metro1),
      metro2_state = pull_state_from_metro(metro2)
    )
}

project_year <- function(pop_df, lambda_df, gamma) {
  pop_vec <- setNames(pop_df$population, pop_df$state_name)
  states <- pop_df$state_name
  
  new_pop <- sapply(states, function(s) {
    in_flow <- lambda_df |>
      filter(state_current == s, state_1y != s) |>
      mutate(flow = lambda * sqrt(pop_vec[state_1y] * pop_vec[s])) |>
      summarize(total = sum(flow, na.rm = TRUE)) |>
      pull(total)
    
    out_flow <- lambda_df |>
      filter(state_1y == s, state_current != s) |>
      mutate(flow = lambda * sqrt(pop_vec[s] * pop_vec[state_current])) |>
      summarize(total = sum(flow, na.rm = TRUE)) |>
      pull(total)
    
    pop_vec[s] * (1 + gamma) + in_flow - out_flow
  })
  
  tibble(state_name = states, population = as.numeric(new_pop))
}

#01: The Case for Expanding the House

The current size of the House of Representatives — fixed at 435 since 1929 — has not kept pace with American population growth. In 1929, each representative served roughly 280,000 constituents. Today that number exceeds 760,000. This dramatic increase means that individual Americans have far less direct representation than their great-grandparents did. A larger House would bring representation ratios closer to those intended by the founders, and would reduce the outsized influence that small population swings have on apportionment outcomes.

From an equity standpoint, the current 435-seat cap creates significant imbalances in population per representative across states. As shown in our apportionment analysis, some states have nearly twice the population per district compared to others. For example, Rhode Island has roughly 490,000 residents per seat while larger states average over 700,000. Expanding the House to 600 or 700 seats would smooth out these inequalities, giving every American more equal representation regardless of which state they live in. The Huntington-Hill method works better with more seats to allocate — rounding errors become less impactful when the total pool is larger.

For New York specifically, a larger House would be advantageous. With a fixed 435 seats, New York’s declining population share directly costs it representation. But in a 600-seat House, New York would likely retain more seats proportionally because there would be more seats available to reflect its large absolute population. A state of 16 million people deserves robust representation — expanding the House is the most straightforward way to ensure that large, diverse states like New York maintain meaningful political power in Congress without relying solely on population growth campaigns.
